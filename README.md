# Leetcode Solutions

This is my repository for all my leetcode solutions, written in Go.

## Repository Structure

```
└── src
    ├── easy
    │   ├── easyProblem
    │   │   └── easySolution.go
    ├── hard
    │   ├── hardProblem
    │   │   └── hardSolution.go
    └── medium
        ├── mediumProblem
        │   └── mediumSolution.go
```

The directories inside the difficulties are named by the leetcode problem's ID number
